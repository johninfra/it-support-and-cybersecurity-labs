## Featured Labs

### Lab 03 - Network Troubleshooting and Connectivity Restoration
Diagnosed and resolved a network connectivity issue by identifying DHCP misconfiguration and restoring proper IP assignment.

### Lab 04 - Active Directory Access Control
Created users, assigned groups, configured shared folders, and applied NTFS permissions in a domain environment.

### Lab 08 - DNS and Network Connectivity Troubleshooting
Simulated DNS failure and restored name resolution using command line tools and network diagnostics.

### Lab 11 - Enterprise Windows Troubleshooting Ticket
Simulated a real world help desk ticket involving a printer failure. Diagnosed the issue using system tools, restarted the Print Spooler service, and verified resolution with a successful test print.

### Lab 12 - Network Traffic Analysis and Monitoring
Captured and analyzed live network traffic using Wireshark and performed network scanning using Nmap to identify active hosts and protocols.
