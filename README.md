# IT Support and Cybersecurity Home Labs

This repository showcases hands on labs designed to simulate real world IT support and cybersecurity scenarios.

## Labs Included

- Network reconnaissance using Kali Linux and Nmap
- Linux system navigation and file permissions
- Network troubleshooting and connectivity repair
- Active Directory user and access control management
- Network drive mapping and file sharing

## Skills Demonstrated

- Network troubleshooting and diagnostics
- Command line proficiency (Linux and Windows)
- Virtualization using VMware
- Active Directory fundamentals
- File system permissions and access control
- Real world help desk problem solving

## Goal

To build practical experience and transition into an entry level IT support or cybersecurity role.
